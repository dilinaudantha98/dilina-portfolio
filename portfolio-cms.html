How to Deploy Free with GitHub Pages

▼
  1. Create a free GitHub account and a new public repository (e.g. my-portfolio)
  2. Upload portfolio.html to the repo root
  3. In repo Settings → Pages → set source to main branch / root
  4. Create a Personal Access Token (classic) with repo scope at GitHub → Settings → Developer settings
  5. Fill in the form above and click Save Settings, then Push Data Now
  6. From now on, clicking 💾 Save Changes in the CMS auto-pushes to GitHub
  7. Your live site is at https://yourusername.github.io/my-portfolio/portfolio.html
